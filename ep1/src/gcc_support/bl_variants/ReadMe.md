bl_variants
===========

This project explores ARM/Thumb Branch Links to provide insights into how GCC compiles ARM/Thumb code.

## Requirements

* `arm-none-eabi-gcc`, `arm-none-eabi-objdump`, `arm-none-eabi-objcopy`

## Build and Run

```bash
# Linux
make -f Makefile.eg1
make ARGON=1 -f Makefile.eg1
```
